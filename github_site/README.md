# Defense Capital Dashboard

A comprehensive dashboard tracking defense sector investments, economic indicators, and market performance.

## Overview

This dashboard combines macro and micro perspectives on the defense capital landscape:

- **Economic Indicators**: 10 Federal Reserve (FRED) series tracking defense spending, industrial production, and investment trends
- **Deal Tracker**: Curated private capital investments in defense companies and projects
- **Market Data**: Defense sector ETFs, industrial equities, and Treasury yields

## Features

- **Interactive Charts**: Hover to see exact values, fully responsive
- **Searchable Deals**: Filter by sector, capital type, or search terms
- **Mobile-Friendly**: Works on all devices
- **Fast Loading**: Static site with optimized assets
- **Professional Design**: Clean teal theme (#226E93)

## Data Sources

- Federal Reserve Economic Data (FRED)
- Yahoo Finance (market data)
- Curated Google Alerts (deal tracking)

## Technology

- Chart.js for interactive visualizations
- Vanilla JavaScript (no heavy frameworks)
- Responsive CSS with modern design
- Python backend for data fetching and HTML generation
